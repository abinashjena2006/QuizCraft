Quiz Craft

Project Name - Online Quiz and Competitive Exam platform

Project Description:
We are doing the Online Quiz and Competitive Exam Platform project to make exams easier, faster, and more convenient for everyone.
In the traditional exam system, a lot of time is wasted on paperwork, manual checking, and result preparation. This often causes delays and sometimes mistakes.
With this project, students can take quizzes online from anywhere, and the system checks the answers automatically and shows results instantly.
It also helps teachers and administrators manage questions and results more easily. Through this project,
we learn how to connect frontend, backend, and database while building a useful system that solves a real-life problem in education.

Technologies Used :

•	Frontend
HTML – structure of pages
CSS – design and layout
JavaScript – basic validation and interaction

•	Backend
Python with Flask (to handle the backend that includes the evaluate answers and etc)

•	Database
MySQL (to store in a database)

Functional Requirements:
	These describe what the system should do.
	Users should be able to register and log in to the system.
	Users should be able to attempt quizzes/exams online.
	The system should display questions with multiple-choice options.
	A timer should be available for each quiz.
	The system should automatically evaluate answers.
	Users should be able to view their results immediately after submission.
	Admin should be able to add, edit, and delete quiz questions.
	The system should store user details, questions, and results in the database.

Non-Functional Requirements:
	These describe how well the system should work.
	The system should be easy to use and user-friendly.
	The application should be fast and responsive.
	The system should be secure so user data is protected.
	The platform should be reliable and work without crashes.
	The system should be easy to maintain and update.

How Our Model Solves the Problem :
	Our platform replaces the manual examination process with a fully automated online system. Students can log in to the platform, select the quiz, and attempt questions within a fixed time limit. Once the quiz is submitted, the system automatically evaluates the answers and displays the result immediately.
	Teachers or administrators can add, update, or remove questions easily without paperwork. All user details, questions, and results are stored securely in a database.
  By using web technologies, the system can be accessed from anywhere.
